Генератор placeholder изображений

Инструкция:

  1. Откройте этот файл в браузере
  2. Нажмите правой кнопкой мыши на каждое изображение
  3. Выберите "Сохранить изображение как..."
  4. Сохраните в папку images/ с соответствующим именем:
    • apartment.jpg
    • house.jpg
    • office.jpg

Или используйте кнопки "Скачать" под каждым изображением.
